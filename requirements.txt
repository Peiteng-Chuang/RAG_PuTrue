# ============================================================================
#  安裝步驟（順序重要）
# ============================================================================
#  1) 先單獨裝 PyTorch（CUDA 綁定，不要靠這份檔從 PyPI 直接裝）
#  2) 再 pip install -r requirements.txt  （torch 已裝好，這步只補其餘套件）
#
# ----------------------------------------------------------------------------
#  PyTorch（GPU 版，必須用對應 CUDA 的官方 index）
# ----------------------------------------------------------------------------
#  基準機（2050）實際版本： torch==2.11.0+cu128 / torchvision==0.26.0+cu128
#  你的 4090 目前是 cu124 —— cu124 對 RTX 4090（Ada, sm_89）完全夠用，不必改成 cu128。
#  依機器的 CUDA build 選 index（兩者要同一 index 一起裝，版本才相容）：
#
#    # 4090（cu124，維持現況即可）
#    pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
#
#    # 若要與基準機完全一致（cu128）
#    pip install torch==2.11.0 torchvision==0.26.0 --index-url https://download.pytorch.org/whl/cu128
#
#  裝完務必驗證抓得到 GPU：
#    python -c "import torch;print(torch.__version__, torch.cuda.is_available(), torch.cuda.get_device_name(0))"
#
